# Monika S ❤️ Madhav J Wedding Invitation

A premium luxury black-and-gold wedding invitation website built with HTML5, CSS3, and vanilla JavaScript.

## Included Features

- Cinematic loading screen with animated golden crest
- Full-screen hero section with particles, overlay, and opening interaction
- Bride and groom profile cards
- Live countdown for 31 August 2026 at 09:15 AM Asia/Kolkata
- Wedding event timeline
- Romantic story cards
- Responsive masonry photo gallery with lightbox
- Venue section with Google Maps link
- RSVP form saved in browser LocalStorage
- Bride and groom contact cards with WhatsApp and call buttons
- Floating hearts, gold sparkles, smooth scrolling, reveal animations, and hover effects
- Music player using `assets/music/background.mp3` with a graceful browser-audio fallback

## How To Use

Open `index.html` in any modern browser. Replace the image files and `assets/music/background.mp3` with final wedding photos and music when ready, keeping the same filenames.

RSVP entries are stored locally in the browser under the key `monikaMadhavRsvps`.
